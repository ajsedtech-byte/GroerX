# GroerX Full Stack Working Code

A production-style starter built from the supplied GroerX documentation and UI references.

## Run Backend
```bash
cd backend
npm install
npm run dev
```
API: http://localhost:5000/api/health

## Run Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend: http://localhost:5173

## Pages
- /dashboard
- /assessment
- /round-result
- /results
- /careers
- /mentor

## Notes
This version includes a working frontend, working Express backend, reusable components, assessment flow, demo scoring engine, career data, responsive UI and routes.
